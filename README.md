# 🧠 Learn HTML Tags

A comprehensive, interactive webpage demonstrating the use of **HTML tags** — from basic text formatting to advanced forms and multimedia. Built as a beginner-friendly resource to **learn**, **visualize**, and **practice** semantic HTML elements.

👉 [Visit Website](https://ramteja-16.github.io/learn-html-tags/)

## 📌 Highlights

- ✅ Headings, paragraphs, bold/italic/emphasized/deleted/inserted text
- ✅ Lists: unordered, ordered, and description
- ✅ Links, images (with alt/title), broken images, clickable images
- ✅ Tables with `rowspan`, `colspan`, headers, and captions
- ✅ Forms with all major input types, radio, checkbox, range, select, textarea
- ✅ Semantic tags: `header`, `nav`, `section`, `article`, `aside`, `footer`
- ✅ Multimedia embedding: video, audio, map via iframe
- ✅ Sticky navbar and responsive layout

## 📁 How to Use

1. Clone this repository:
   ```bash
   git clone https://github.com/ramteja-16/learn-html-tags.git
   ```
2. Open `index.html` in any web browser.

---

## 📤 Deploy Your Own

To host this site using GitHub Pages:

1. Push your files to a GitHub repository.
2. Go to **Settings > Pages**
3. Choose:
   - Branch: `main`
   - Folder: `/ (root)`
4. Save and visit the generated GitHub Pages URL.
